# Website Name: VoyageVista

## Description
This project is a Tourist Spots Management System implemented using MERN stack (MongoDB, Express.js, React.js, Node.js). It allows users to view, add, and delete tourist spots. Users can register/login using Firebase Authentication to manage their own spots.

## Technologies Used
- MongoDB: Database for storing tourist spot data and user information.
- Express.js: Backend framework for building RESTful APIs.
- React.js: Frontend library for building user interfaces.
- Node.js: JavaScript runtime environment for running server-side code.
- Firebase Authentication: For user authentication and authorization.
- React Router: For client-side routing.
- Cors: Middleware for enabling Cross-Origin Resource Sharing.
- React Simple Typewriter: Used for animating text in the banner section.
- React Awesome Reveal: Used for animating elements on scroll.

## Main Components
- **Backend**: Implemented using Node.js and Express.js, responsible for handling HTTP requests, interacting with the database, and serving API endpoints.
- **Frontend**: Implemented using React.js, responsible for rendering the user interface, managing state, and making requests to the backend APIs.
- **Database**: MongoDB used for storing data related to tourist spots and user accounts.
- **Authentication**: Firebase Authentication used for user registration and login.
- **Tourist Spot Management**: Users can view, add, and delete tourist spots.
- **Responsive Design**: The application is designed to be responsive, ensuring a seamless user experience across different devices.

## Libraries and Packages Used
- React Simple Typewriter: Used for adding typewriter effect to text in the banner section.
- React Awesome Reveal: Used for animating elements on scroll, providing a visually appealing user experience.

## Installation and Setup
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using `npm install`.
4. Start the server using `npm start`.


## Usage
- Register/Login: Users can register and login using Firebase Authentication to access the application.
- View Spots: Users can view a list of tourist spots.
- Add Spot: Logged-in users can add new tourist spots.
- Delete Spot: Logged-in users can delete tourist spots they have added.
